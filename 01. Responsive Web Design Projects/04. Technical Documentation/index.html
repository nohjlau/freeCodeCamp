    • concat

    The concat() method is used to join two or more arrays. This method does not change the existing arrays, but returns a new array containt the values of the joined arrays.

    var fruits = ['apple', 'bananan', 'orange']; var vegetables = ['lettuce', 'spinach']; var fruitsAndVegetables = fruits + vegetables;
    constructor

    The constructor property returns the constructor function for an object.

    var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.constructor;
              • copyWithin

              The copyWithin() method copies array elements to another position in the array, overwriting the existing values, but will never add more items to an array.

              var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.copyWithin(2, 0); //var fruits = ["Banana", "Orange", "Banana", "Orange"];
                        • entries

                        The entries() method returns an Array Iterator object with key/values pairs.

                        var fruits = ["Banana", "Orange", "Apple", "Mango"]; var f = fruits.entries(); for (x of f) { document.getElementById("demo").innerHTML += x; }
                                  • every

                                  The every() method checks if all elements in an array pass a test provided as a function)
                                  The every() method executes the function once for each element present in the array:

                                  • If it finds an array element where the function returns a false value, every() returns false (and does not check the remaining values)
                                    If no falses occur, every() returns true
                                  • This could be useful if you want to check if every element in an array is for example a number. var ages = [32, 33, 16, 40]; function checkAdult(age) { return age >= 10; } function myFunction() { document.getElementById("demo").innerHTML = ages.every(checkAdult); }
